# OC_Roboc_Multiplayer
Jeu de labyrinthe multijoueur.
Programme Python réalisé dans le cadre des cours OpenClassrooms

Pour jouer, (après avoir installé Python),
  - démarrer le serveur
  - y sélectionner une carte
  - démarrer un client par joueur
  - suivre les instructions
  - une fois la partie commencée, taper 'H' pour afficher la liste des commandes
